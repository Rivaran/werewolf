# スマホGM人狼

スマートフォンを使って人狼ゲームの進行を行うためのGM補助アプリです。

夜フェーズ、役職行動、朝アナウンス、議論タイマーなどを自動で進行できます。

## 主な機能

・夜フェーズ進行  
・人狼襲撃  
・占い結果表示  
・騎士護衛  
・朝アナウンス音声  
・議論タイマー  
・勝敗判定

## 使用技術

Next.js  
React  
TypeScript
